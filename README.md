# Hardware-Hackathon-EA2

Explanation of our project from hardware hackathon.

For the 2026 hardware Hackathon, my partner and I were tasked to build a device that fit the theme 'Seamless Living'. My partner and I came up with a idea of a device that tracks if a person leave a stove on, if so they will be alerted to turn off the stove. My role of this project was to determine the tools and parts we need and configuring the hardware.
